# 🎬 Movie App
 
A React-based web application for browsing popular movies and managing a personal favorites list, powered by the [TMDB (The Movie Database) API](https://www.themoviedb.org/).
 
![React](https://img.shields.io/badge/React-19-61DAFB?logo=react) ![React Router](https://img.shields.io/badge/React_Router-7-CA4245?logo=react-router) ![Vite](https://img.shields.io/badge/Vite-6-646CFF?logo=vite) ![TMDB](https://img.shields.io/badge/TMDB-API-01B4E4?logo=themoviedatabase)
 
## Features
 
- 🔥 Browse popular movies fetched from TMDB
- 🔍 Search for any movie by title
- ❤️ Add and remove movies from your favorites
- 💾 Favorites persist across sessions via `localStorage`
- 📱 Responsive movie grid layout

## Tech Stack
 
| Tool | Purpose |
|------|---------|
| React | UI library |
| React Router DOM | Client-side routing |
| React Context API | Global favorites state |
| TMDB API | Movie data & images |
| localStorage | Favorites persistence |
| Vite | Build tool & dev server |
